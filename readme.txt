QR Code Multi Purpose Widget to display QR code with your details on your sidebar

== Description ==

QR Code Multi Purpose Widget to display QR code with your details on your sidebar

Needs Curl enabled

Options:

1. Bookmark - Will provide title and URL 

2. Business card - Will provide title, address, phone and email id

3. Call a phone - Will provide phone number, can be called from your device

4. URL - provide url so that visitors can open the website in their device

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to widgets page

4. Drag and drop the widget where ever you need


== Frequently asked questions ==

1. Which bar code reader will read this?

All QR code readers can scan and use the details 


2. The image is not opening properly

Needs Curl enabled
